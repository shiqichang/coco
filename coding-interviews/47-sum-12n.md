## 题目描述

求1+2+3+...+n，要求不能使用乘除法、for、while、if、else、switch、case等关键字及条件判断语句（A?B:C）。

## 题解

```python
def sum(n):
    """ 求和公式 """
    return (1 + n) * n // 2
```

```python
def sum(n):
    """ 递归短路 """
    return n and (n + sum(n-1))
```

🍥 **考察要点**：逻辑运算符

🍬 **解题思路**：求和公式；递归短路代替 if 语句

🍉 **时间复杂度**：O(n)

🍭 **空间复杂度**：O(1)
