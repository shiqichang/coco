## 题目描述

给定一个整数，写一个函数来判断它是否是 3 的幂次方。

### 示例1

> 输入: 27
> 输出: true

### 示例2

> 输入: 0
> 输出: false

### 示例3

> 输入: 9
> 输出: true

### 示例4

> 输入: 45
> 输出: false

### 进阶

你能不使用循环或者递归来完成本题吗？

## 题解

```python
def is_power_of_three(n):
    """ 循环迭代 """
    if n < 1:
        return False
    while n % 3 == 0:
        n //= 3
    return n == 1
```

```python
def is_power_of_three(n):
    """ 整数限制 """
    return n > 0 and 1162261467 % n == 0
```

🍥 **考察要点**：迭代

🍬 **解题思路**：循环迭代，`n % 3`只要余数是0，就一直将 `n` 除以 `b` ；基准转换；运算法；整数限制，时间复杂度为 O(1)。

🍉 **时间复杂度**：O(logn)

🍭 **空间复杂度**：O(1)
