# 05 有效的括号

> 🌈 **初级算法系列之其它**
>
> 你的无畏源于无知。

## 题目描述

给定一个只包括 `'('`，`')'`，`'{'`，`'}'`，`'['`，`']'` 的字符串，判断字符串是否有效。
有效字符串需满足：

1. 左括号必须用相同类型的右括号闭合。
2. 左括号必须以正确的顺序闭合。

注意空字符串可被认为是有效字符串。

### 示例1

> 输入: "()"
> 输出: true

### 示例2

> 输入: "()[]{}"
> 输出: true

### 示例3

> 输入: "(]"
> 输出: false

### 示例4

> 输入: "([)]"
> 输出: false

### 示例5

> 输入: "{[]}"
> 输出: true

## 题解

```python
def is_valid(s):
    if len(s) % 2 == 1:
        return False
    
    pairs = {
        ")": "(",
        "]": "[",
        "}": "{"
    }
    stack = []
    for ch in s:
        if ch in pairs:
            if not stack or stack[-1] != pairs[ch]:
                return False
            stack.pop()
        else:
            stack.append(ch)
    return not stack
```

🍥 **考察要点**：栈

🍬 **解题思路**：栈，后遇到的左括号要先闭合。

🍉 **时间复杂度**：O(n)

🍭 **空间复杂度**：O(n)
